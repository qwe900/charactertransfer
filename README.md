# Character Transfer Module
!!!!WIP not finished yet. WIP !!!!!
This module allows users to transfer their World of Warcraft characters from external servers to the current realm using a character dump addon.

## Features

- Upload character dumps generated by the Chardump addon
- Admin panel to review and approve/deny transfers
- Automatic item replacement for unavailable items
- 3D character model viewer (need node proxy for this to work)
- Achievement and inventory display
- Talent specialization viewing

## Installation

1. Ensure the module is placed in `application/modules/charactertransfer/`
2. Run the SQL files on your fusion database to create necessary tables:
   - `character_transfer.sql`
   - `character_transfer_item_replacements.sql`
3. Access the module via the admin panel to configure permissions (not finished yet)

## Usage

### For Users

1. Download and install the Chardump addon for World of Warcraft
2. Log into your character, open your bank, and type `/chardump` to generate a dump
3. Upload the generated `.lua` file via the character transfer page
4. Wait for admin approval

### For Admins

- View pending transfers in the admin panel
- Review character details, including equipment, achievements, and stats
- Approve or deny transfers
- Modify item replacements if needed

## Requirements

- FusionGen v2
- World of Warcraft Wrath of the Lich King (3.3.5a)
- Chardump addon (this is also not finished yet)

## Security Notes

- All user inputs are validated
- Database queries use prepared statements
- External API calls are handled with error checking
